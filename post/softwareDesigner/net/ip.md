# ip类别

|  类别   |         ip地址区域          | 网络位：主机位 |
| :-----: | :-------------------------: | :------------: |
|   A类   |  0.0.0.0 ~ 127.255.255.255  |      1：3      |
|   B类   | 128.0.0.0 ~ 191.255.255.255 |      2：2      |
|   C类   | 192.0.0.0 ~ 223.255.255.255 |      3：1      |
| D类组播 | 224.0.0.0 ~ 239.255.255.255 |                |
| E类保留 | 240.0.0.0 ~ 255.255.255.255 |                |

## 子网划分（一变多）

### 产生原因

ip地址的类别分配太过严格，难以按照实际需求分配合理的地址空间
因此产生子网验码
