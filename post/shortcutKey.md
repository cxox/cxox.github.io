# 快捷键

## Windows

### 全局

| 快捷键             | 功能               |
| ------------------ | ------------------ |
| ctrl + a           | 全选               |
| ctrl + c           | 复制               |
| ctrl + x           | 剪切               |
| ctrl + z           | 撤销               |
| ctrl + shift + esc | 任务管理器         |
| ctrl + tab         | 应用内部标更换签页 |
| alt + tab          | 更换应用           |
| win                | 开始屏幕           |
| win + tab          | 任务视图           |
| win + e            | 文件资源管理器     |
| win + i            | 系统设置           |
| win + v            | 剪切板             |
| win + .            | 表情符             |
